# VirtualLan

ZeroTier liked system

Features:

- DHCP Sevrer build in
- UDP channel between two clients

Requirement:
- Tap Driver installed (tap-windows-9.24.2-I601-Win10.exe)

Usage of vlan:
  -port int
        Local Port (default 1900)
  -remote string
        Destination Server:Destination Port
